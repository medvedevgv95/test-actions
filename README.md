# Time Server API

Простое FastAPI приложение для получения текущего времени сервера.

## Описание

Это веб-сервис на базе FastAPI, который предоставляет API для получения текущего времени сервера. Приложение имеет два эндпоинта для работы со временем.

## Возможности

- Получение текущего времени сервера в формате ISO
- Получение timestamp текущего времени
- RESTful API с автоматической документацией

## Технологии

- **FastAPI** - современный веб-фреймворк для Python
- **Uvicorn** - ASGI сервер для запуска приложения
- **Python 3.11** - версия Python
- **Docker** - контейнеризация приложения

## Установка и запуск

### Локальный запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python main.py
```

Или через uvicorn напрямую:
```bash
uvicorn main:app --host 0.0.0.0 --port 8000
```

### Запуск через Docker

1. Соберите Docker образ:
```bash
docker build -t time-server-api .
```

2. Запустите контейнер:
```bash
docker run -p 8000:8000 time-server-api
```

## API Эндпоинты

### GET `/`
Возвращает текущее время сервера и сообщение об успешном выполнении.

**Ответ:**
```json
{
  "current_time": "2024-01-01T12:00:00.000000",
  "message": "Server time retrieved successfully"
}
```

### GET `/time`
Возвращает время сервера в формате ISO и timestamp.

**Ответ:**
```json
{
  "server_time": "2024-01-01T12:00:00.000000",
  "timestamp": "1704110400.0"
}
```
